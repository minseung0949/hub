중급재무회계 3장: 회계시스템 (핵심 요약)

제1절: 복식부기시스템

모든 거래를 차변(왼쪽)과 대변(오른쪽)으로 나누어 기록하며, 차변의 합계와 대변의 합계는 반드시 일치해야 합니다.

1. 회계등식 (The Accounting Equation)

자산 = 부채 + 자본

  • 자산 (Asset): 기업이 소유한 경제적 자원 (예: 현금, 건물, 매출채권)
  • 부채 (Liability): 기업이 갚아야 할 의무 (채권자지분)
  • 자본 (Capital): 자산에서 부채를 뺀 순수한 소유주지분

2. 계정과목의 5대 범주

모든 거래는 이 5가지 계정 중 하나로 분류됩니다. 각 계정은 증가하거나 감소하는 위치(차변/대변)가 정해져 있습니다.

계정 범주 정의 (무엇인가?) 차변 (왼쪽) 대변 (오른쪽) 평상시 잔액 위치
자산 (Asset) 기업이 소유한 것 (재산) 증가 (+) 감소 (-) 차변
부채 (Liability) 기업이 갚아야 할 것 (빚) 감소 (-) 증가 (+) 대변
자본 (Capital) 자산-부채 (순자산) 감소 (-) 증가 (+) 대변
비용 (Expense) 돈을 쓴 것 (자본의 감소) 증가 (+) 감소 (-) 차변
수익 (Revenue) 돈을 번 것 (자본의 증가) 감소 (-) 증가 (+) 대변

[암기 Tip!]

  • "자산"과 "비용"은 차변(왼쪽)에서 증가합니다.
  • "부채", "자본", "수익"은 대변(오른쪽)에서 증가합니다.

3. 예제 3-1: 대차균형의 원리 적용하기

거래 내용 분개 (차변 / 대변) 회계등식의 변화 (자산 = 부채 + 자본)
① 용역 제공, 대금 ₩1,000은 나중에 받기로 함 (차) 매출채권 1,000 (자산 증가)
(대) 용역수익 1,000 (수익 발생→자본 증가)		 +1,000 = 0 + 1,000
② 매출채권 ₩500을 현금으로 회수 (차) 현금 500 (자산 증가)
(대) 매출채권 500 (자산 감소)		 +500 - 500 = 0 + 0
③ 주식 발행으로 현금 ₩5,000 조달 (차) 현금 5,000 (자산 증가)
(대) 자본금 5,000 (자본 증가)		 +5,000 = 0 + 5,000
④ 복사기 ₩2,000 구입, 어음 지급 (차) 비품 2,000 (자산 증가)
(대) 미지급금 2,000 (부채 증가)		 +2,000 = +2,000 + 0
⑤ 미지급금 ₩300을 현금으로 지급 (차) 미지급금 300 (부채 감소)
(대) 현금 300 (자산 감소)		 -300 = -300 + 0
⑥ 인건비 ₩100을 현금으로 지급 (차) 인건비 100 (비용 발생→자본 감소)
(대) 현금 100 (자산 감소)		 -100 = 0 - 100

제2절: 회계순환과정

거래가 발생한 시점부터 재무제표가 작성되고 장부가 마감되기까지의 전체 흐름입니다.

  1. (1단계) 거래의 식별과 기록
  2. (2단계) 분개 (분개장에 기록)
  3. (3단계) 원장으로의 전기
  4. (4단계) 수정분개 (★매우 중요★)
  5. (5단계) 재무제표의 작성
  6. (6단계) 장부마감

(1)~(3) 거래 식별, 분개, 전기

거래를 식별하여(1단계) 분개장에 차변/대변으로 기록하고(2단계), 총계정원장(T-계정)에 옮겨 적어 잔액을 집계합니다(3단계).

(4) 수정분개 (Adjusting Entries)

발생주의 회계원칙에 따라, 실제 현금 수수와 관계없이 발생한 수익과 비용을 해당 기간에 정확히 인식하기 위해 기말에 수행하는 분개입니다.

수정분개 대상 항목

  1. 이연 항목 (Deferrals): 현금은 먼저 오고 갔으나, 수익/비용 인식은 나중에 하는 것.
    • 선급비용: 비용을 먼저 지급했으나, 아직 사용하지 않은 부분. (예: 선급임차료)
      (수정분개) → (차) OOO비용 (사용한 만큼) / (대) 선급비용
    • 선수수익: 수익을 먼저 받았으나, 아직 용역을 제공하지 않은 부분. (부채)
      (수정분개) → (차) 선수수익 / (대) OOO수익 (제공한 만큼)
  2. 발생 항목 (Accruals): 수익/비용은 먼저 발생했으나, 현금은 나중에 오고 가는 것.
    • 미지급비용: 비용이 발생했으나, 아직 지급하지 않은 부분. (예: 미지급급여)
      (수정분개) → (차) OOO비용 / (대) 미지급비용
    • 미수수익: 수익이 발생했으나, 아직 받지 못한 부분. (예: 미수이자수익)
      (수정분개) → (차) 미수수익 / (대) OOO수익
  3. 기타 항목 (추정 및 평가)
    • 추정 항목: 감가상각비, 대손상각비 등.
    • 평가 항목: 자산/부채의 기말 공정가치 평가.

(5) 재무제표의 작성

수정후잔액시산표의 자료를 이용하여 재무제표를 작성합니다.

  • 포괄손익계산서: 수익과 비용 계정을 옮겨 당기순이익을 계산.
  • 재무상태표: 자산, 부채, 자본 계정을 옮김. (이때 당기순이익은 자본의 이익잉여금에 합산)

(6) 장부마감 (Closing Entries)

기말에 회계기간을 마감하기 위해 임시계정(수익, 비용, 배당금 등)의 잔액을 "0"으로 만들고, 그 순액(당기순이익)을 영구계정(자산, 부채, 자본)인 이익잉여금으로 대체하는 과정입니다.

마감 분개 순서

  1. 모든 수익(대변 잔액)을 (차변)으로, 집합손익(대변)으로 보냅니다.
  2. 모든 비용(차변 잔액)을 (대변)으로, 집합손익(차변)으로 보냅니다.
  3. 집합손익 계정의 잔액(당기순이익)을 이익잉여금으로 보냅니다.
  4. (기타포괄손익을 기타포괄손익누계액으로 마감)
  5. 배당금(차변 잔액)을 이익잉여금(차변)에서 직접 차감합니다.

→ 이 과정이 끝나면 모든 수익/비용 계정은 0이 되고, 자산/부채/자본 계정만 남아 다음 기로 이월됩니다.
